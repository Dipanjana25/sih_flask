# SIH backend
## How to get started:
> create a virtual environment:
```
python3 -m venv venv
```
>Install the required libraries:
```
pip install -r requirements.txt
```
> Run flask app:
```
flask --app app run
```
> Visit the browser on link given and search query

> # Important:
* Add venv (virtual environment) file to .gitignore file
